# Car Sales Report Analysis Using SQL.

This project is based on Car Sales Report data set downloaded from kaggle.
- Dataset Link- https://www.kaggle.com/datasets/missionjee/car-sales-report
- Business Question 
- Monthly unit sales & revenue trend (seasonality, ASP)
- Dealer & region performance leaderboard (units, revenue, ASP)
- Sales mix & price dispersion by body style
- Top models and their regional performance
- Transmission preferences by gender & region
- Engine type impact on price & popularity per company
- Color price premium after controlling for model
- Income segment penetration (quartiles) & affordability ratio
- Model-level price outliers (z-score / IQR approach)
- RFM segmentation (Recency, Frequency, Monetary) per customer (phone)
